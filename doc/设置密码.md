## 空密码设置密码

```sh
# 5.7.6 以前
mysql> set password for 用户名@localhost = password('新密码');

# 5.7.6 以后
mysql> set password for 用户名@localhost = '新密码';

# 8.0 以后
mysql> ALTER USER 'root'@'localhost' IDENTIFIED BY '新密码';
```
